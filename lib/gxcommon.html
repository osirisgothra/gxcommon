NAME

gxcommon - the gxcommon library

VERSION

Version 0.01

SYNOPSIS

The gxcommon library contains all of the shared libraries used among various paradisim projects. It is not restricted to any one language, and at the time of this writing pertains to at the least, C++ and of course, perl code.

EXPORT

None
